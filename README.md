
# こまない.com — 雛形（Next.js + Supabase + Turnstile + Leaflet）

## 必要な環境変数（Vercel で設定）
- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY
- NEXT_PUBLIC_TURNSTILE_SITE_KEY
- TURNSTILE_SECRET_KEY
- NEXT_PUBLIC_MAPTILER_KEY
- NEXT_PUBLIC_GA_ID (任意)

## ローカル開発（任意）
```bash
npm install
npm run dev
```
